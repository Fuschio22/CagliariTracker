# <img src="https://upload.wikimedia.org/wikipedia/commons/9/9c/Cagliari_Calcio_1920.svg" alt="Cagliari" width="40" height="40" style="vertical-align: middle;"> Cagliari Tracker

App per seguire tutte le partite del **Cagliari Calcio** in Serie A.

## Funzionalità

- 📅 **Calendario completo** - Tutte le partite del Cagliari
- 🏆 **Classifica Serie A** - Con il Cagliari evidenziato
- ⏰ **Countdown** - Tempo alla prossima partita
-  **Risultati recenti** - Ultime 5 partite con esito (V/P/S)
- 🔔 **Notifiche** - Avviso 1 ora prima di ogni partita

## Tecnologia

- HTML, CSS, JavaScript
- API: football-data.org
- Hosting: GitHub Pages
