# MatchPulse

Real-time football live match tracker with modern UI.

## Features

- Live match tracking
- Real-time score updates
- Event notifications (goals, cards, substitutions)
- Match center with statistics
- Team search
- Favorites system
- PWA support (installable)
- Dark theme with glassmorphism design

## Live Demo

https://fuschio22.github.io/MatchPulse/

## Technologies

- HTML5
- CSS3 (Glassmorphism, Animations)
- Vanilla JavaScript
- football-data.org API
- GitHub Pages

## How to Use

1. Open the live demo link
2. View featured match
3. Click on any match for detailed stats
4. Search for teams
5. Add matches to favorites

## API

This project uses [football-data.org](https://www.football-data.org/) API for real-time match data.
